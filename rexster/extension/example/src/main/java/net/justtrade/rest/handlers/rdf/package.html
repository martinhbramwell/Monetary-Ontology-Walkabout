Just Trade is an initiative towards an equitable money system able to facilitate the activity of the productive sector without channeling the lions share of the benefit to the financial sector. This package contains all the software that collaborators have created towards that goal.

Package Specification
At this point there exists no overview specification for this package.
My document {@link Productive Money vs. Financial Money} is an early vision statement.
